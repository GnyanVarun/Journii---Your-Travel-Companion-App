# 🌍 Journii

> An AI-powered travel companion that combines intelligent trip planning, live event discovery, immersive mapping, and personalized recommendations into a single seamless travel experience.

## ✨ The Vision

Journii was built around a simple idea:

Travel should feel effortless, personalized, and intelligent.

Instead of manually searching across multiple apps for attractions, routes, local events, restaurants, and recommendations, Journii acts as a digital travel companion that understands your destination, interests, and available time.

By leveraging AI, real-time data, and location intelligence, Journii helps travelers:
<ul>
  <li>Discover hidden gems</li>
  <li>Plan optimized day itineraries</li>
  <li>Explore local events</li>
  <li>Navigate efficiently</li>
  <li>Learn about places through conversational AI</li>
  <li>Create memorable experiences instead of managing logistics</li>
</ul>

## 🚀 Key Features

### 🧠 AI Travel Assistant

Powered by Google Gemini, Journii provides natural language travel assistance including:
<ul>
  <li>Destination recommendations</li>
  <li>Personalized travel suggestions</li>
  <li>Conversational trip planning</li>
  <li>Attraction insights</li>
  <li>Context-aware travel guidance</li>
</ul>

### 🗺️ Smart Interactive Maps
Built using Mapbox and OpenStreetMap data.

Features Include:
<ul>
  <li>Interactive map exploration</li>
  <li>Custom route generation</li>
  <li>Turn-by-turn navigation</li>
  <li>Place discovery</li>
  <li>Nearby attraction lookup</li>
  <li>Dynamic route previews</li>
</ul>

### 🎫 Live Event Discovery
Journii aggregates real-time events from multiple providers.

Discover:
<ul>

  <li>Sports events</li>
  <li>Community gatherings</li>
  <li>Theatres</li>
</ul>

Data Sources Include:
<ul>
  <li>PredictHQ</li>
</ul>

### 📍 Intelligent Place Exploration
Explore destinations with rich contextual information.

<ul>
